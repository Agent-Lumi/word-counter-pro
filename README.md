# 📝 Word Counter Pro

A powerful, beautiful word counting tool that goes beyond simple word counts. Get detailed text analytics including character count, reading time estimates, and keyword density analysis — all in a clean, modern interface.

![Word Counter Pro Screenshot](screenshot.png)

## 🚀 Live Demo

**Try it now:** [https://agent-lumi.github.io/word-counter-pro/](https://agent-lumi.github.io/word-counter-pro/)

## ✨ Features

- **📊 Comprehensive Stats**: Word count, character count (with/without spaces), sentence count, paragraph count
- **⏱️ Reading Time**: Estimated reading time based on average reading speed
- **🔍 Keyword Density**: Automatically analyzes and displays the most frequently used words
- **🎨 Beautiful Dark Theme**: Easy on the eyes with a modern, clean design
- **📱 Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **⚡ Fast & Lightweight**: No dependencies, 100% client-side processing
- **🔒 Privacy First**: Your text never leaves your browser — no server uploads
- **💾 Works Offline**: Use it anywhere, even without internet

## 🎯 How to Use

### Online (Recommended)
Simply visit the [live demo](https://agent-lumi.github.io/word-counter-pro/) and start typing or paste your text!

### Local Installation
```bash
# Clone the repository
git clone https://github.com/Agent-Lumi/word-counter-pro.git

# Navigate to the directory
cd word-counter-pro

# Open in your browser
open index.html
```

### Usage Tips
1. **Paste or type** your text into the editor
2. **Watch stats update** in real-time as you type
3. **View keyword density** to optimize your content
4. **Use reading time** to gauge content length for your audience
