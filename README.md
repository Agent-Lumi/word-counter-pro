# 📝 Word Counter Pro

Advanced text analysis tool with reading time & keyword density tracking.

## ✨ Features

- **📊 Text Statistics**: Words, characters, sentences, paragraphs
- **⏱️ Reading Time**: Based on 200 WPM average
- **🔑 Keyword Density**: Top 10 keywords with bar charts and list views
- **🌙 Dark Mode**: Toggle between light and dark themes
- **⌨️ Keyboard Shortcuts**: Power-user friendly
- **📤 Export Results**: TXT, JSON, or CSV formats
- **💾 Auto-Save**: localStorage persistence
- **📱 PWA Support**: Install as a standalone app

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + Enter` | Analyze text |
| `Ctrl + S` | Save text |
| `Ctrl + Shift + C` | Copy text |
| `Ctrl + Shift + X` | Clear text |
| `Ctrl + D` | Toggle dark mode |
| `?` | Show shortcuts help |
| `Esc` | Close modals |

## 🚀 Usage

1. Paste or type your text in the textarea
2. Click "Analyze Text" or press `Ctrl+Enter`
3. View statistics, reading time, and keyword density
4. Export results or save text for later

## 🛠️ Tech Stack

- Vanilla HTML/CSS/JavaScript
- No dependencies
- PWA with service worker
- LocalStorage for persistence
